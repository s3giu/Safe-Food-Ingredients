# Safe-Food-Ingredients

A lightweight, cloud-based web application that analyzes food labels for harmful additives using Computer Vision and LLMs.

Features:

AI Analysis: Uses Google Gemini 1.5 Flash to perform OCR (text extraction) and toxicological assessment in a single pass.

Instant Verdicts: Flags ingredients related to cancer, metabolic disruption, and EU bans.

Secure Access: Protected by a custom password system to prevent unauthorized public usage.

Zero-Cost Deployment: Runs entirely on Streamlit Community Cloud with Google's Free Tier API.

Tech Stack: Python, Streamlit, Google Generative AI (Gemini), Pillow.
